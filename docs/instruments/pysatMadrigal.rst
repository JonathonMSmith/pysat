pysatMadrigal
-------------

`pysatMadrigal <https://github.com/pysat/pysatMadrigal>`_ provides pysat
support for data available at the
`Madrigal database. <http://cedar.openmadrigal.org/index.html>`_
See the
`Supported Instruments <https://pysatmadrigal.readthedocs.io/en/latest/supported_instruments.html>`_
page for more information.
